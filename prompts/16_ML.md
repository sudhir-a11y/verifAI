# Doctor + Auditor ML Model — Full Development Plan

## Goal

Train ML model using **Doctor + Auditor decisions** to auto-predict:

- approve
- reject
- query
- manual_review

---

# What Model Learns

Inputs:

```text
ai_decision
ai_confidence
risk_score
verification_flags
rule_hits
conflicts
structured_data
```

Labels (from humans):

```text
doctor_decision
auditor_decision
final_decision
```

Target:

```text
final_decision
```

---

# Data Source

Tables:

```text
claims
decision_results
doctor_verifications
auditor_verifications
document_extractions
```

# Training Dataset

Build rows:

```text
features → label
```

```text

```
